Hero

DaffHeroComponent is a top level container that is large and captivating. It should only be used once as the first container on any given page. It supports transcluding any content and optionally including a predefined title and subtitle.

Supported Content Types

A <daff-hero> transcludes:
  • [daffHeroTagline]
  • [daffHeroTitle]
  • [daffHeroSubtitle]
  • Any additional components

Tagline

Hero tagline is used by adding daffHeroTagline to a <p> or <div> tag.

Title

Hero title is used by adding daffHeroTitle to a <h1> tag.

Subtitle

Hero subtitle is used by adding daffHeroSubtitle to a <h2> tag.

Theming

The hero's background color is defined by using the color property. By default, the color is set to theme-contrast. This can be changed to one of the supported colors.

Supported colors: primary | secondary | tertiary | black | white | theme | theme-contrast

Size

The size property will be deprecated in v1.0.0. The compact value will be replaced by the DaffCompactable interface.

Layout

The layout property will be deprecated in v1.0.0
