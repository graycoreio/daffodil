Image

The image component is used to customize an image you want to display in your project. To those familiar with the HTML img, daff-image should feel similar. daff-image is an opinionated version that encourages end-user friendly usage.

Usage

You can modify the width, height, alt, and src of the image component.

Sizes

The size of an image can be updated by using the height and width of the property. Any size is supported in pixels. Height and width are required to display the image.
