Variables

Daffodil's CSS custom properties can be used for fast and easy design and development. We provide common, functional CSS variables that can be used to quickly style elements without the need to create theme files.

CSS Custom Properties

Our custom properties are prefixed with daff- to avoid conflicts with third party variables.

Variable Description
--daff-theme-rgb The theme's base color in RGB
--daff-theme-contrast-rgb The theme's base contrast color in RGB
--daff-theme The theme's base color
--daff-theme-contrast The theme's base contrast color
--daff-theme-primary The theme's primary color
--daff-theme-secondary The theme's secondary color
--daff-theme-tertiary The theme's tertiary color
--daff-theme-warn The theme's warning color
--daff-theme-success The theme's success color
--daff-theme-danger The theme's danger color
