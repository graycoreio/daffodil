Daff Paginator

Daffodil's paginator component can be used to navigate between pages of content.

About

The paginator can be used to break up large amounts of content and enable the user to easily navigate the data. The paginator also helps with ease of use by displaying the number of pages as well as links for traversal to the "previous" or "next" page.

Theming

Setting color on the paginator component will change the default background color of the selected page and the hovered page.

Supported colors: primary | secondary | tertiary | black | white | theme | theme-contrast

Accessibility

Use aria-label or aria-labelledby to label the daff-paginator component.
